📊 Netflix Data Analysis

This project is an interactive Power BI dashboard built to analyze and visualize content data from Netflix. The report provides insights into content distribution, trends, genres, ratings, release patterns, and global availability using dynamic visualizations and data modeling techniques.

🔎 Project Overview

The goal of this project is to explore Netflix’s content library and uncover meaningful insights through data analytics and business intelligence tools. The dashboard transforms raw data into actionable information, allowing users to:

Analyze the distribution of Movies vs TV Shows

Explore content trends by year of release

Examine genre popularity

Identify top contributing countries

Understand content ratings distribution

Discover growth patterns over time

The project demonstrates practical skills in data cleaning, transformation, modeling, and visualization using Power BI.

🛠 Tools & Technologies Used

Power BI Desktop (.pbix file)

Power Query (Data Cleaning & Transformation)

DAX (Data Analysis Expressions)

Data Modeling & Relationships

Interactive Visualizations & Filters

📈 Key Dashboard Features

Content Type Analysis – Breakdown of Movies vs TV Shows

Release Year Trends – Visual representation of content growth over time

Genre Insights – Most common and trending categories

Country Distribution – Geographic contribution to Netflix’s catalog

Rating Classification – Audience targeting analysis

Interactive Filters – Dynamic slicers for deeper exploration

📂 Dataset Description

The dataset includes attributes such as:

Title

Director

Cast

Country

Date Added

Release Year

Rating

Duration

Genre (Listed In)

Content Type (Movie/TV Show)

Data preprocessing involved handling missing values, standardizing categories, and creating calculated columns/measures for enhanced analytics.
